# Vue

## 1. 环境搭建

​		※注意： 命令在Windows环境下需要去掉开头的 `sudo` 。

​		1）全局安装 `vue-cli` 

````shell
sudo npm install -g vue-cli
````

​		2）使用 `webpack` 模板构建项目

````
vue init webpack [项目名称]

实例：
vue init webpack mall
````

​		3）安装项目依赖

````shell
sudo npm install
````

​		4）以开发环境运行项目

````shell
sudo npm run dev
````



# Css

## 1. 开发规范

​		1）不可继承的属性尽量写在前面（如 `display、width、height` 等），可被继承的属性尽量写在后面（如 `font-size、color` 等）

````css
.tab
    display: flex
    width: 100%
    height: 40px
    line-height: 40px
    .tab-item
      flex: 1
      text-align: center
      & > a
        display: block
        font-size: 14px
        color: rgb(77, 85, 93)
````

# 2. 样式笔记

​		1）触摸区域撑满

````css
	display: block
````

​		2）去掉下划线

````css
	text-decoration: none;
````
