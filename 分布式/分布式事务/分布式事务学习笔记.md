# 第一章 事务原则与实现

## 1.1 事务

**1.概念**

​		事务是一种以**可靠、一致**的方式，**访问和操作**数据库中数据的**程序单元**。

**2. 原则**

- 原子性
- 一致性
- 隔离性
- 持久性

**3. 实现**

​		1）mysql中：sql语句为

````mysql
BEGIN TRANSACTION;
UPDATE t_user amount = amount - 100 WHERE username = '张三';
UPDATE t_user amount = amount + 100 WHERE username = '李四';
COMMIT
-- ROLLBACK
````
